*******************************************************************************
**  This component requires sh404SEF to be isntalled to function correctly.  **
*******************************************************************************

To use:
1. install this component.
2. in sh404SEF configure a New URL where SEF URL is desired "old" address and Non-SEF URL is the following:
index.php?component=com_externalredirect&redirect=http://www.example.com
(http://www.example.com should be replaced with the address of the page you want to redirect to).
